# AWS Task Manager

## 📌 Project Description
This is a simple serverless Task Manager web application built using AWS services. It allows users to add tasks through a web interface, which are stored in a backend database using AWS DynamoDB.


## 🚀 Features
- Add new tasks from web UI
- Store tasks in AWS DynamoDB
- Serverless backend using AWS Lambda
- API integration using AWS API Gateway


## 🛠️ Tech Stack
- HTML (Frontend)
- JavaScript (inside HTML)
- AWS Lambda (Python)
- AWS API Gateway
- AWS DynamoDB


## 🏗️ Architecture
Frontend (HTML) → API Gateway → Lambda → DynamoDB


## 📡 API Endpoints
- POST /create → Add a new task
- GET /tasks → Retrieve all tasks


## 💡 Key Learning
- Serverless architecture using AWS
- API integration with frontend
- Working with DynamoDB database
- Building cloud-based web applications
